# Bib-Cleaner
A Python utility to clean bibliography files by removing 'unnecessary' fields such as URL, ISSN, ISBN, and DOI.

## Usage
To use the Bib-Cleaner, run the following command in your terminal:
```bash
bib-cleaner path/to/your/bibliography.bib --backup t/f
```
